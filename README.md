# Beyond Darkness

A dark reskin of the classic Doodle Jump, built with C++ and SFML.

## Concept

you play a fallen angel who's trying to return in heaven from the hell.
## Features

- atmosphère inspired by hollow knight
- 4 platform types: Normal, Moving, Breakable, Bouncy
- Enemy system with static and moving threats
- A wave mechanic that pursues the player
- Score-based progression with a win condition at 30 000 points

## Controls

| Key | Action |
|-----|--------|
| ← → | Move left / right |
| Arrow keys | Resist the wave when grabbed |
| Space | Restart (on win screen) |
| Escape | Quit (on win screen) |

## Built With

- C++20
- SFML 3.0.2

## How to Run

1. Clone the repository
2. Open the solution in Visual Studio
3. Build in Release or Debug mode
4. Run the executable — make sure the `images/` folder is next to it
